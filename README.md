# SafePass - Secure Identity & Digital Vault

Welcome to the **SafePass-Public** repository for Android releases! This repository hosts the official APK downloads for the SafePass Android application.

## 🚀 The Core Problem & Our Solution
**The Problem:** Managing passwords, sensitive documents, and tracking multiple subscriptions securely is difficult. Data breaches, lost offline backups, and complicated interfaces make identity protection a headache.  
**The Solution:** SafePass acts as a fully encrypted digital vault. It works flawlessly in **Offline Mode** to guarantee true privacy and offers **Online Cloud Sync** so your securely encrypted data seamlessly follows you across devices.

## 📱 App Highlights & Modules

* **🔑 Password Page:** Securely store, manage, and autofill your passwords. Built-in leak detection and strong password generation keep your accounts safe.
* **📄 Document Page:** Store sensitive files, contracts, and IDs securely. Includes an advanced AI Document Scanner with edge-detection and an encrypted PDF Editor.
* **💳 Subscription Page:** Stop paying for unused services! Track active subscriptions, monitor costs, and get priority alerts before billing cycles hit.
* **🧑‍💻 API Key Page:** Designed for developers. Securely stash and organize your development tokens, keys, and secrets without friction.

## ✨ Core Features
* **📶 Offline & Online:** True offline mode means your vault works entirely without an internet connection until you explicitly want to sync. Background cloud sync handles the rest.
* **🔓 Biometric Access:** Native Fingerprint & Face Unlock right out of the box.
* **💰 Free & Pro Tiers:** The critical security features and offline vault are completely free forever. Pro tiers give you advanced cloud automation and premium perks.
